# BMC-GOV Website Revamp Project Plan

## Project Overview
Complete revamp of bmcgov.com (Burdwan Medical College and Hospital) website using Next.js 16, Shadcn/ui, with modern UI/UX, maintaining existing functionality and content.

**Repository:** bmc-gov  
**Framework:** Next.js 16.2.1 with App Router  
**UI Library:** Shadcn/ui (installed)  
**Styling:** Tailwind CSS v4  
**Theme:** Healthcare-focused (Blue, White, Grey palette) with Light/Dark mode

---

- **White:** #FFFFFF
- **Light Grey:** #F5F7FA

### Accent Colors
- **Success Green:** #10B981 (for positive actions)
